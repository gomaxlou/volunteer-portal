團務報告系統使用手冊

系統概述

團務報告系統是一個專為志工團隊設計的會議記錄管理平台。本系統提供直觀的介面，方便使用者查看、搜尋和下載會議記錄。

主要功能：

  • 會議記錄瀏覽與搜尋
  • PDF 格式下載
  • 中文（繁體）完整支援
  • 響應式設計，支援各種設備

使用指南

1. 瀏覽會議記錄

在左側面板中，您可以看到所有會議記錄的列表：

  • 依時間順序排列，最新的會議記錄顯示在最上方
  • 每個項目顯示會議標題和日期
  • 點擊任何會議項目，右側會顯示詳細內容

提示：使用滾輪或觸控板可以快速瀏覽長列表

2. 搜尋功能

系統提供強大的搜尋功能，可以快速找到需要的會議記錄：

  • 在頂部搜尋框輸入關鍵字
  • 支援多個關鍵字組合搜尋
  • 可搜尋範圍：
    • 會議標題
    • 會議日期
    • 會議內容

搜尋技巧：

  • 使用空格分隔多個關鍵字
  • 搜尋結果會即時顯示
  • 系統會自動標記符合的內容

3. PDF 下載

系統支援將會議記錄轉換為 PDF 格式下載：

  • 點擊右上角的「下載 PDF」按鈕
  • 系統會自動生成包含完整會議內容的 PDF 文件
  • PDF 文件包含：
    • 會議標題和日期
    • 完整會議內容
    • 格式化的段落和標題

注意事項：

  • PDF 生成可能需要幾秒鐘時間
  • 請確保瀏覽器允許下載檔案
  • 建議使用最新版本的瀏覽器

登入系統

1. 登入頁面

系統採用帳號密碼認證方式：

  • 在登入頁面輸入您的帳號和密碼
  • 系統會驗證您的身份
  • 登入成功後會自動導向到團務報告頁面

登入說明：

  • 帳號和密碼區分大小寫
  • 如果忘記密碼，請聯絡系統管理員
  • 為了安全起見，請定期更換密碼

2. 登出功能

您可以透過以下方式登出系統：

  • 點擊頁面右上角的「登出」按鈕
  • 系統會清除您的登入狀態
  • 自動導回登入頁面

安全提醒：

  • 離開電腦時請記得登出
  • 不要在公共電腦上儲存密碼
  • 使用完畢請關閉瀏覽器

3. 帳號安全

為了確保系統安全，請遵守以下準則：

  • 密碼長度至少 8 個字元
  • 密碼應包含英文字母和數字
  • 不要使用容易猜到的密碼
  • 不要與他人共享帳號

重要提醒：

  • 系統登入狀態會在 24 小時後自動過期
  • 請勿在不安全的網路環境登入
  • 如發現帳號異常請立即通知管理員

活動管理

1. 活動列表

在活動列表頁面，您可以：

  • 查看所有活動的基本資訊
  • 使用搜尋功能快速找到特定活動
  • 點擊活動卡片查看詳細資訊
  • 管理員可以編輯或刪除活動

2. 新增活動

管理員可以透過新增活動功能來建立新的志工活動：

  • 必填欄位：
    • 活動名稱
    • 開始日期
    • 結束日期
    • 地點
    • 最大參與人數
    • 報名截止日期
    • 專案經理資訊（姓名、Email、電話）
  • 選填欄位：
    • 活動圖片
    • 專案經理職稱
    • Line ID
    • 類別
    • 難度
    • 需求事項
    • 福利
    • 攜帶物品
    • 注意事項
    • 交通資訊
    • 集合地點
    • 行程表

注意事項：

  • 報名截止日期必須早於活動開始日期
  • 最大參與人數必須大於 0
  • 專案經理 Email 必須符合正確格式
  • 電話號碼必須至少 8 位數

3. 編輯活動

管理員可以編輯已建立的活動：

  • 在活動列表中點擊「編輯」按鈕
  • 修改需要更新的欄位
  • 點擊「儲存」完成編輯

編輯提醒：

  • 編輯時需要重新驗證所有必填欄位
  • 如果不修改圖片，將保留原有圖片
  • 編輯完成後會自動返回活動列表

系統需求

支援的瀏覽器

  • Google Chrome (推薦)
  • Mozilla Firefox
  • Microsoft Edge
  • Safari

建議的螢幕解析度

  • 桌面：1920 x 1080 或更高
  • 平板：1024 x 768 或更高
  • 手機：支援任何解析度

常見問題

Q: 為什麼我的搜尋沒有結果？

A: 請確認：

  • 檢查關鍵字拼寫是否正確
  • 嘗試使用較短或較常見的關鍵字
  • 確保沒有輸入特殊符號

Q: PDF 下載失敗怎麼辦？

A: 可以嘗試以下解決方案：

  • 確認網路連線穩定
  • 重新整理頁面後再試
  • 檢查瀏覽器是否封鎖下載
  • 使用其他支援的瀏覽器

Q: 系統支援哪些語言？

A: 目前系統支援：

  • 繁體中文（預設）
  • 所有 PDF 內容皆支援中文字型

聯絡支援

  • 填寫系統回饋表
  • 向團務組組長反應
  • 在週會中提出建議
