主要功能模組

  • 活動管理（/events）
    • 活動建立與編輯
    • 報名管理
    • 活動執行追蹤
    • 結案報告
  • 相簿功能（/gallery）
    • 照片上傳與管理
    • 相簿分類
    • 照片瀏覽
    • 分享功能
  • 志工管理（/volunteer）
    • 志工資料管理
    • 服務時數統計
    • 證書管理
  • 報告系統（/reports）
    • 活動報告生成
    • 統計分析
    • 資料匯出

技術堆疊

前端框架

  • Next.js 15.0.3
  • React 18.2.0
  • TypeScript 5.x
  • Tailwind CSS
  • MUI 5.x

後端技術

  • Python 3.10
  • FastAPI
  • SQLAlchemy

資料庫

  • PostgreSQL
  • Redis（快取）

開發工具

  • Git
  • Docker
  • VS Code
