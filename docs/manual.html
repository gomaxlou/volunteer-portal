寶島志工服務系統 - 使用手冊

版本 1.0.0 | 最後更新：2024年1月

系統使用手冊

系統概述

寶島志工服務系統是一個專為志工團隊設計的綜合管理平台。本系統提供活動管理、志工服務時數統計、報告生成等功能，幫助團隊更有效地管理志工服務工作。

主要功能：

  • 活動管理（建立、編輯、查詢）
  • 志工服務時數統計
  • 報告生成與下載
  • 用戶管理與權限控制

功能說明

1. 活動管理

活動管理功能提供完整的志工活動生命週期管理：

  • 活動建立
    • 設定活動名稱、日期、地點
    • 指定活動類型和服務內容
    • 設定參與人數上限
  • 活動編輯
    • 修改活動詳情
    • 更新活動狀態
    • 管理報名名單
  • 活動查詢
    • 依日期、類型搜尋
    • 檢視活動詳情
    • 匯出活動報告

2. 志工服務管理

系統提供完整的志工服務時數管理：

  • 服務時數登記
    • 選擇活動並登記時數
    • 記錄服務內容
    • 上傳相關證明
  • 時數統計
    • 個人服務時數查詢
    • 團隊服務時數統計
    • 服務類型分析

3. 報告功能

系統提供多種報告查看和下載功能：

  • 統計報告
    • 活動參與統計
    • 服務時數統計
    • 志工貢獻分析
  • PDF 下載功能
    • 報告轉PDF下載
    • 支援中文字體
    • 自動分頁處理

注意：

  • 報告頁面為公開頁面，無需登入即可查看
  • 建議使用最新版本的瀏覽器
  • 大型報告可能需要較長時間生成

4. 用戶管理

系統提供完整的用戶管理功能：

  • 用戶註冊與登入
    • 建立個人帳號
    • 登入系統
    • 密碼重置
  • 權限管理
    • 管理員權限
    • 一般用戶權限
    • 訪客權限

系統需求

支援的瀏覽器

  • Google Chrome (推薦)
  • Mozilla Firefox
  • Microsoft Edge
  • Safari

建議螢幕解析度

  • 桌面：1920 x 1080 或以上
  • 平板：1024 x 768 或以上
  • 手機：支援所有解析度
