活動管理 API

GET /api/events

獲取活動列表

查詢參數：

參數名 類型 必填 說明
page number 否 頁碼，預設為 1
limit number 否 每頁數量，預設為 10
POST /api/events

創建新活動

請求體：

{
  "title": "string",
  "description": "string",
  "startDate": "string (YYYY-MM-DD)",
  "endDate": "string (YYYY-MM-DD)",
  "location": "string",
  "maxParticipants": "number",
  "contact": {
    "name": "string",
    "email": "string",
    "phone": "string"
  }
}

用戶管理 API

POST /api/auth/login

用戶登入

請求體：

{
  "email": "string",
  "password": "string"
}
POST /api/auth/register

用戶註冊

請求體：

{
  "name": "string",
  "email": "string",
  "password": "string",
  "phone": "string"
}

相簿管理 API

GET /api/gallery

獲取相簿列表

POST /api/gallery/upload

上傳相片

請求體：

使用 FormData 格式

  • image: File
  • description: string
  • eventId: number

報告管理 API

GET /api/reports

獲取報告列表

POST /api/reports/generate

生成活動報告

請求體：

{
  "eventId": "number",
  "reportType": "string",
  "format": "pdf|docx"
}
