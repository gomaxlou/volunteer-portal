套件版本變更記錄

更新日期：2024/12/03

版本衝突問題

發現的衝突：

  • googletrans 3.0.0 需要 httpx==0.13.3，但安裝的是 httpx 0.27.2
  • mediapipe 0.9.1.0 需要 protobuf<4,>=3.11，但安裝的是 protobuf 4.22.1
  • open-clip-torch 2.16.0 需要 protobuf<4，但安裝的是 protobuf 4.22.1

解決方案

已執行的更改：

  • 降級 protobuf 到 3.20.3 以支援 mediapipe 和 open-clip-torch
  • 更新 httpx 到最新版本以支援 openai 和 zhipuai
  • 移除 googletrans 並安裝 py-googletrans

React 相關套件更新

套件版本更改：

  • react: 19.0.0-rc-66855b96-20241106 → ^18.2.0
  • react-dom: 19.0.0-rc-66855b96-20241106 → ^18.2.0
  • @emotion/react: ^11.13.5 → ^11.11.3
  • @emotion/styled: ^11.13.5 → ^11.11.0
  • @mui/material: ^6.1.9 → ^5.15.11

注意事項

  • 若使用 grpcio-status 相關功能，請特別測試，因為它需要較新版本的 protobuf
  • 已將 React 相關套件更新到穩定版本，避免使用 RC（候選發布）版本可能帶來的不穩定性
  • 如遇到任何功能異常，請回報問題
