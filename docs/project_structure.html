專案結構說明文件

版本 1.0.0 | 最後更新：2024年1月

專案根目錄
package.json
專案配置文件，定義了： - 專案依賴項及其版本 - 開發和部署腳本 - 專案基本信息 - 開發工具配置
next.config.js
Next.js 框架配置文件，包含： - 路由重寫規則 - 環境變數配置 - 圖片優化設置 - API 路由配置
tsconfig.json
TypeScript 配置文件，定義： - 編譯選項 - 模組解析規則 - 類型檢查設置 - 路徑別名配置
tailwind.config.js
Tailwind CSS 配置文件，包含： - 自定義主題設置 - 顏色方案定義 - 響應式斷點配置 - 自定義插件設置
src/
app/
Next.js 13+ 的頁面路由目錄，包含所有頁面組件和路由配置。
api/
API 路由處理目錄，包含所有後端 API 端點的實現。
auth/route.ts
處理用戶認證相關的 API 請求，包括登入、註冊等功能。
events/route.ts
處理活動相關的 API 請求，包括創建、更新、刪除活動等功能。
about/
關於我們頁面，介紹組織理念和目標。
reports/
報表和統計功能相關頁面。
layout.tsx
Next.js 根布局組件，定義了整個應用的基本結構和共享元素。
page.tsx
首頁組件，展示主要功能和最新活動。
events/
活動管理相關的頁面，包含列表、詳情、編輯等功能。
auth/
使用者認證相關頁面，包含登入、註冊等功能。
components/
features/
功能性組件，實現特定業務邏輯的UI組件。
EventCard.tsx
活動卡片組件，用於展示活動概要信息。
EventForm.tsx
活動表單組件，用於創建和編輯活動。
layout/
布局相關組件，包括頁頭、頁尾、導航欄等。
Header.tsx
頁面頂部導航欄組件，包含logo和主要導航鏈接。
Footer.tsx
頁面底部組件，包含聯繫方式和相關鏈接。
shared/
共用UI組件，如按鈕、輸入框等基礎組件。
lib/
db/
events.ts Database
活動相關的資料庫操作函數和類型定義。

相關檔案：

  • dbOperations.ts - 基礎資料庫操作
  • edit/page.tsx - 活動編輯頁面
  • technical_flow.html - 資料庫結構文檔
users.ts Database
用戶相關的資料庫操作函數和類型定義。

相關檔案：

  • auth.ts - 認證相關功能
  • dbOperations.ts - 基礎資料庫操作
  • technical_flow.html - 資料庫結構文檔
auth.ts Auth
認證相關的核心功能，包括登入、註冊、權限驗證等。

相關檔案：

  • users.ts - 用戶資料庫操作
  • api/auth/route.ts - 認證 API 端點
  • middleware.ts - 認證中間件
dbOperations.ts Database
資料庫操作的核心功能，包含所有資料庫交互的基礎實現。

相關檔案：

  • events.ts - 活動資料庫操作
  • users.ts - 用戶資料庫操作
  • technical_flow.html - 資料庫結構文檔
constants.ts
全域常數定義，包括配置項、狀態碼等。
types.ts
全域類型定義，包括共用介面和類型。
types/
TypeScript 類型定義文件。
public/
images/
靜態圖片資源目錄，包含網站所需的所有圖片。
default-event.jpg
活動的預設圖片，當活動沒有上傳圖片時使用。
icons/
網站使用的圖標文件目錄。
scripts/
db-init.ts
資料庫初始化腳本，用於創建資料庫表格和初始數據。
seed.ts
測試數據生成腳本，用於填充測試環境的示例數據。
volunteer.db
SQLite 資料庫文件，儲存所有應用數據： - 用戶資料 - 活動信息 - 報名記錄 - 系統配置
docs/
專案文檔目錄，包含技術文檔和使用說明。
project_structure.html
專案結構說明文件，描述專案目錄結構和相關檔案。
technical_flow.html
技術流程文檔，描述系統的技術實現和流程。
user_manual.html
使用者手冊，描述系統的使用方法和注意事項。
faq.html
常見問題解答，描述系統的常見問題和解決方法。
scripts/
工具腳本目錄，包含資料庫遷移、測試等腳本。

頁面結構

  • / - 首頁
    • 展示志工服務概況
    • 提供快速導航到各功能區
  • /login - 登入頁面
    • 用戶認證
    • 權限管理
  • /reports - 報告頁面（公開訪問）
    • 志工服務統計報告
    • 支援 PDF 下載功能
    • 自動分頁處理
    • 支援中文字體
  • /events - 活動管理
    • 活動列表
    • 活動詳情
    • 報名功能
  • /donation - 捐款頁面
    • 捐款信息
    • 捐款方式說明

權限管理

  • 公開頁面（無需登入）：
    • 首頁 (/)
    • 登入頁面 (/login)
    • 關於我們 (/about)
    • 捐款頁面 (/donation)
    • 報告頁面 (/reports)
  • 需要登入訪問：
    • 活動管理 (/events/*)
    • 用戶管理 (/users/*)
    • 管理面板 (/admin/*)
