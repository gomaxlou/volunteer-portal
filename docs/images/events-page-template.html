2024春季淨灘活動

活動日期 2024-03-15
活動地點 新北市八里區觀海廣場
報名截止 2024-03-10
名額限制 50人

活動說明

為了維護海洋生態環境，我們將在八里海灘舉辦淨灘活動。活動除了淨灘外，也會安排海洋生態教育講座，歡迎大家一同參與。

活動流程

09:00-09:30 報到集合
09:30-10:00 活動說明及分組
10:00-11:30 淨灘活動
11:30-12:00 海洋生態講座
12:00-13:00 午餐交流

注意事項

1. 請穿著適合戶外活動的服裝
2. 建議攜帶防曬用品、帽子
3. 現場提供手套及清潔工具
4. 活動提供午餐及飲用水
